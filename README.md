## ✨ Objetivo del repositorio

## 🗂 Estructura general de carpetas

La estructura principal del repositorio es la siguiente:

- `PARCIAL1_SOL/` — Contiene los archivos y soportes del primer parcial (scripts, datos, generadores de PDF, gestor de DB, notebooks, etc.).
- `series-temporales/` — Contiene datos y notebooks relacionados con series temporales y análisis de series (por ejemplo: `exchange_rates.csv` y notebooks de prueba).


> Nota: La estructura puede ampliarse conforme avancemos en la materia.

---

## 📌 Cómo se mantendrá este repositorio

- Todas las evidencias (códigos, datos, notebooks, informes y PDFs) se dejarán dentro de las carpetas correspondientes.
- Cada entrega o avance importante irá acompañado de un pequeño `README.md` dentro de su carpeta explicando archivos, pasos para ejecutar y resultados esperados.
- Usaremos control de versiones con commits claros y mensajes descriptivos.

## 🤝 Buenas prácticas
